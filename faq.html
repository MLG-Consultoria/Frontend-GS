Perguntas frequentes

Nosso projeto é uma solução baseada em IoT (Internet das Coisas), que utiliza sensores para monitorar variáveis ambientais (temperatura, umidade do ar, CO₂, entre outros) e detectar riscos de incêndios florestais de forma antecipada. Os dados são analisados em tempo real, e alertas são enviados para usuários e autoridades.
Atualmente monitoramos:
  • Temperatura do ar
  • Umidade relativa do ar
  • Níveis de gases (como CO₂)
  • Velocidade do vento (opcional)

    • Essas variáveis ajudam a identificar se o ambiente está propenso a queimadas.

    Os dados são coletados por sensores instalados em áreas estratégicas. Esses sensores capturam informações ambientais e enviam para um servidor central onde são analisadas.
    Sim. Quando os dados indicam um risco elevado, o sistema envia alertas em tempo real para os usuários cadastrados e pode também notificar órgãos de proteção ambiental e defesa civil.
    Os alertas são acessados por meio de:
  • Aplicativo mobile (em desenvolvimento)
  • Painel web
  • Chatbot no Telegram (com suporte a texto e voz)
    • Sim. Utilizamos IA para classificar os níveis de risco com base nos dados recebidos. Além disso, um chatbot foi desenvolvido para ajudar os usuários com dúvidas e instruções em situações de risco.
    O sistema pode ser implantado em:
  • Áreas florestais
  • Parques urbanos
  • Propriedades rurais
  • Regiões com histórico de queimadas
    • Nosso objetivo é criar uma solução de baixo custo, com sensores simples e tecnologia de código aberto, para que possa ser replicada em diferentes comunidades.
    Estamos estudando alternativas como:
  • Comunicação via rádio (LoRa)
  • Armazenamento local com envio periódico
  • Integração com redes móveis (GSM) ou Wi-Fi comunitário
