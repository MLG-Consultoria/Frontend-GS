Bem-vindo ao Projeto Contra Queimadas

Sobre o Projeto – Guardião Verde

O projeto utiliza dispositivos IoT para monitorar em tempo real fatores como temperatura, umidade e gases de combustão, identificando riscos de incêndio. Os dados são processados por algoritmos que emitem alertas para moradores e autoridades. Além disso, um chatbot via Telegram e web permite consultas sobre o risco na região, dicas de prevenção e orientações em emergências. A iniciativa promove prevenção e educação ambiental por meio de uma plataforma acessível e de baixo custo. A solução é escalável e pode ser aplicada em diversas regiões vulneráveis.

Tecnologias Utilizadas

Sensores ambientais (IoT)

Ánalise de dados em tempo real

Chatbot com IBM Watson Assistant

Integração com Webchat e Telegram (STT e TTS)

Front-end responsivo em HTML/CSS/JS

Aplicações em Python e Java

Banco de dados relacional
