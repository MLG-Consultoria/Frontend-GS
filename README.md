# Projeto Guardião Verde - Global Solution 

## Objetivo do Projeto:

O nosso objetivo é **prevenir queimadas e reduzir seus impactos por meio de uma solução tecnológica acessível e inteligente.** Através do uso de sensores IoT e algoritmos de análise de risco, o projeto busca monitorar o ambiente em tempo real e fornecer alertas para a população e autoridades. Além disso, promove a educação ambiental com um chatbot que orienta sobre prevenção, emergência e consequências das queimadas, contribuindo para a proteção do meio ambiente e a segurança das comunidades vulneráveis.
